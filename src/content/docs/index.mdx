## Why FlightManual?

FlightManual is a completely open-source, edge-ready documentation framework designed to go toe-to-toe with enterprise platforms like Mintlify and ReadMe. Built on Astro Starlight, it brings a premium developer experience, integrated API playgrounds, and a fully customizable Shadcn UI component library to your documentation.

<CardGrid stagger>
	<Card title="Mintlify-Tier DX" icon="setting">
		Stunning UI with custom Shadcn primitives like `<ApiField>`, `<Endpoint>`, and `<CodeGroup>` that look phenomenal out of the box.
	</Card>
	<Card title="Docs-as-Code" icon="document">
		Manage your documentation natively in Git. Use our Scramjet content pipeline to auto-generate docs directly from Zod schemas and source code.
	</Card>
	<Card title="Interactive API Playground" icon="rocket">
		Integrated Scalar OpenAPI playgrounds. Allow your developers to test endpoints, view payloads, and generate `cURL` snippets instantly.
	</Card>
	<Card title="AI-Native Outputs" icon="star">
		Auto-generates `llms.txt` and `sitemap.xml` on every build. Your documentation is instantly readable by Perplexity, ChatGPT, and Claude.
	</Card>
	<Card title="Stateful AI Agent" icon="bot">
		A native, Llama 3 powered `assistant-ui` Chatbot built on Cloudflare Durable Objects. Zero SaaS fees.
	</Card>
	<Card title="Cloudflare Edge RBAC" icon="lock">
		Protect enterprise documentation directories with native Edge Middleware. Complete access control.
	</Card>
</CardGrid>
