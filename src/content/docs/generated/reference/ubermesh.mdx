Polymorphic content-addressed substrate. `defineType` generates full API surfaces. The pipeline IS the CMS.

## What Is This

UberMesh is the unified content layer (Layer 1) for the Atlas ecosystem. You declare types. UberMesh makes them real:

```typescript
defineType("post", {
  title: "string",
  body: "rich_text",
  brand: { ref: "brand" },
  surface: "string",
})
// -> REST endpoints, MCP tools, CLI commands, TypeScript types, query validation
```

Every piece of content is the output of a transformation pipeline, stored content-addressed. Content compounds -- every output becomes input to the next transformation.

## Principles

1. **Content compounds.** Every output can be input to the next pipeline.
2. **The pipeline IS the CMS.** AI produces. Humans supervise.
3. **A brand IS the DAG.** The complete graph of all content transformations.
4. **Polymorphic types, not built-in types.** `defineType()` makes anything real.
5. **Knowledge compounds.** The Karpathy LLM Wiki principle applied to all content.
